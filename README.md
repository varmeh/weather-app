# Weather App

Weather graphs for US Cities

### Tech Stack

- Reactjs
- Redux
- Bootstrap
- CSS3

Uses a custom design of Redux store.
Provision for separate dev & production middleware.

### Libraries

- Axios
- React-sparklines
- Redux-thunk
- Chai
- Mocha
- Webpack
- JSdom

### Running the app

```
> npm install
> npm start
```

Post that Click [here](http://localhost:8080/).
